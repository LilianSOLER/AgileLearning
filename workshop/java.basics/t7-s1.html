The code we gave you runs until you stop it. It loops over a very basic behavior. It creates a circle, represented as a polygon with only four points. Over time, the number of points is incremented so to produce a smoother circle, up to a certain limit. When that limit is reached, it starts all over again. The Graphical User Interface simply renders the circle, as it grows smoother and smoother. Le code que nous vous avons donné s'exécute jusqu'à ce que vous le stoppiez. Il crée un cercle, représenté par un polygone avec seulement quatre points. Au fil du temps, le nombre de points est incrémenté de manière à produire un cercle plus régulier, jusqu'à une certaine limite. Quand cette limite est atteinte, tout recommence. L'interface graphique (GUI) affiche le cercle, montrant son évolution.

Launch the execution and enjoy. See circles getting smoother as they are represented with more and more lines. Look at the top of the window, you are given the number of points in the polygon. This is a well-know technique to represent curves and surfaces in graphic programs. The more points, the more expensive it is to manage geometrical shapes, but the smoother the graphics.

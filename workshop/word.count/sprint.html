sprint=word.count

This sprint is about the famous word count program, available in almost all Linux distribution, as the wc program. It is a simple command line tool that prints the count of lines, words, and characters for the text files given as arguments. Have a quick look at the man page: Ce sprint concerne le fameux programme word-count, disponible sur la plupart des distributions Linux (programme "wc"). C'est un simple outil disponible en ligne de commande, qui affiche le nombre de lignes, mots et caractères pour des fichiers textes passés en arguments. Regardez rapidement la page man du programme wc: 

  $ man wc

We will extend the word-count program with the ability to count the number of occurrences of words. In other words, we will know which words appeared in a text file and how many times they appeared. Vous allez étendre le programme word-count avec la capacité de compter le nombre d'occurences de chaque mot. Autrement dit, nous saurons quels mots apparaissent dans un fichier et combien de fois.

Reminders:

  • Do not forget to import the word.count project in your Eclipse workspace.
  • Do not forget to make sure the tests.jar is part of your build path and that it appears in the Referenced Libraries

Tasks:

Streams are an important concept when programming, this task is about the core idea and the basics of reading or writing streams. In this task, you will learn how to write a very simpler parser of character streams, counting characters, words, and lines. The equivalent of the Linux tool "wc". This task is improving on the previous one. Our word-count program will now count the number of occurrences of each word, which will drastically change the structure of the program. It will be an interesting endeavor, guaranteed! This task is about introducing performance counters in our word count program to evaluate performance and identify potential improvements.

sprint=ricm3.ovm

This project, called OVM that stands for Object Virtual Machine, is about coding an interpreter for a programming language whose syntax is close to the syntax of the Java programming language. The focus will be on the understanding of the mechanisms underlying a programming language, such as variables or functions. The idea is that if you know how a tool works, you can better use it. At the end, you get to write your own programs in your own language and execute on your own runtime, how cool is that? Le projet, appelé OVM pour Object Virtual Machine, est de faire un interpréteur d’un langage proche de celui de Java. Le focus sera sur la compréhension des mécanismes d’un langage de programmation, tels que les variables ou les fonctions. Lorsque l’on sait comment un outil fonctionne, on peut mieux l’utiliser. Cerise sur le gateau: vous pourrez écrire vos programmes dans ce langage et les exécuter avec votre propre plateforme d'exécution. C'est cool, non?

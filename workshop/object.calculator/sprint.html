sprint=object.calculator

This sprint is about creating a calculator, available from the command line. It is not difficult to code and to get it working, but the code can be used to illustrate the use of many of the object-oriented programming concepts we introduced earlier. So keep this in mind, the goal is not just to code a calculator that just works, it is to learn how to write effective object-oriented code in the process. To reach that goal, we will go through several versions of the code. Le but de ce sprint est la création d'une calculatrice en mode ligne de commande. Ce n'est pas du code difficile à faire, mais ce code peut être utilisé pour illustrer de nombreuses façettes de la programmation objets que nous avons introduit lors des sprints précédents. Aussi, il est important de garder à l'esprit que le but de ce sprint n'est pas de juste écrire du code pour avoir une calculatrice qui marche, mais bien d'apprendre à être efficace avec les concepts avancés de la programmation orientée-objet.

You will need to import the project "object.calculator" in Eclipse. It will show some compilation errors, you need to add to the build path the jar file that packaged your collections, the one you created at the end of the sprint "object.collections". Vous allez devoir importer le projet "object.calculator" dans Eclipse. Il ne va pas compiler immédiatement, il va vous falloir ajouter l'archive de vos collections que vous avez créé à la fin du sprint "object.collections".

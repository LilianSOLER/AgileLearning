Browse the prog.basics project to find the Java package ricm3.oop.games. You can see three games: HangedMan, and TicTacToe. The class GameMain is the entry point, with the method main that allows to choose which game to play.

Read and understand the class GameMain, it is an easy read. It illustrates how you can setup a simple command-line menu and ask for the end user to input his choice. As you can see, it expects certain methods on each game. You job is to implement those methods. You will notice that, for the TicTacToe games, it is possible for two human players to play but it also possible for one human player against the machine. Lisez et comprenez la classe GameMain, elle est facile à comprendre. Elle illustre comment mettre en place un simple menu, demandant à l'utilisateur final d'entrer un choix. Comme vous pouvez le voir, certaines méthodes sont attendues pour chaque jeu. Votre travail est d'implémenter ces méthodes. remarquez que, pour le jeu TicTacToe, il est possible de jouer à deux joueurs mais également de jouer à un joueur contre la machine.

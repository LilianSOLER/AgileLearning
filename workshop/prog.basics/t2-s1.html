If you haven't done so already, launch Eclipse, manually, from a shell, as explained earlier.

Switch to the Java perspective, if you are not already there. You can check on the upper-right corner that you are indeed in the Java perspective, with the icon selected in the perspective switcher: Placez vous dans la perspective Java, si vous n'y êtes pas déjà. Vous pouvez vérifier cela en regardant dans le coin en haut à droite de la fenêtre d'Eclipse. Vous êtes dans la perspective Java si l'icon est sélectionnée dans le navigateur de perspective:

Under the Java perspective, you have the Package Explorer view on the left-hand side of the window. We will import an existing Java project that we already setup for you. To import a project, right-click somewhere in the Package Explorer view and select import. Follow the popup dialog, selecting General and then Existing Projects in the workspace. Click on "Next". Sous la perspective Java, vous avez la vue Package Explorer sur le coté gauche de la fenêtre. Vous allez importer un projet Java existant que nous avons déjà configuré pour vous. Pour importer un projet, faites un clic droit quelquepart dans la vue Package Explorer et sélectionnez import. Suivez le popup, sélectionnez General puis Existing Projects in the workspace. Cliquez sur "Next".

Then choose the location of the root directory (~/AgileLearning/workshops/oop/workspace). For now, only import the prog.basics project into your Eclipse workspace, so deselect all projects (Deselect All) and only select the prog.basics project. Click "Finish". Ensuite choisissez la localisation du répertoire racine de votre workspace (~/AgileLearning/workshops/oop/workspace). Pour l'instant n'importez que le projet prog.basics in Eclipse, donc dé-sélectionnez tous les projets (Deselect All) et sélectionnez seulement le projet prog.basics.

Note that the project prog.basics now appears in the Package Explorer view, on the left-hand side of the Eclipse workbench. Notice that you can see the source folder "src". Just ignore the tests.jar for now. You are all set for the next step. Notez que le projet prog.basics apparait maintenant dans la vue Package Explorer, sur le coté gauche du workbench Eclipse. Notez que vous pouvez voir le folder source "src". Ignorez tests.jar pour l'instant. Vous êtes prêts pour la prochaine étape.

sprint=prog.basics

This sprint is about getting you setup and started with your development environment. There is not about programming per se, it is about being ready to become an effective developer. Nevertheless, we will revisit the basic constructs used in programming, while learning how to use the Eclipse tool and especially the Eclipse debugger for Java. So if you are a beginner, you will be able to catch up, but it will require extra work on your part. Ce sprint est pour vous lancer avec votre environement de développement. Il n'est pas encore question encore de programmation en tant que telle, mais nous allons discuter de points importants pour être un développeur efficace. Néanmoins, nous allons revoir les bases de la programmation en apprenant à utiliser le debugger Java. Donc, si vous êtes débutant en programmation, vous allez pouvoir apprendre, mais cela va vous demander un certain inverstissement.

Everything you need to know to compile and execute a Java program, from the command line. We will introduce the basic packaging concept of Java: the source file for a Java class and the directory hierarchy for Java packages. We will illustrate with an HelloWorld class. Java is an imperative programming language, like Python or JavaScript, and it has very similar constructs. This task will review these constructs, using the opportunity to get you even more familiar with Java debugging in Eclipse. It is time that you start writing your own programs, isn't it?

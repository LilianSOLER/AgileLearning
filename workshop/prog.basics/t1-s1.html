Now, let's compile the source we just created. Indeed, Java is a compiled programming language, which means that the Java runtime cannot understand the source of a Java class such as the one we just typed. You need to compile your source into a Java class file. Let's do it: Maintenant, nous allons compiler le source que nous venons de créer. En effet, Java est un language de programmation qui nécessite la compilation des sources avant de pouvoir les exécuter.

See, the Java compiler produced one Java class file, with the ".class" extension. Note that the source and the class file have the same name, only the extension changes. The ".java" extension for sources, the ".class" extension for compiled class files. Notice that a class file is not an executable, from a Linux perspective (it does not have the r-x rights). Vous voyez, le compilateur Java a produit un fichier class file, avec l'extension ".class". Notez que le fichier source et class ont le même nom, seule l'extension change. L'extension pour les sources est ".java" et ".class" pour les fichiers classes. Notez aussi que les fichiers classes ne sont pas des exécutables, au sens Linux du terme.

See, that was easy. You need to memorize these steps:

For your information, the Java runtime is sometimes called the Java Runtime Environment (JRE) or the Java Virtual Machine (JVM). There are subtil distinctions as to what those names exactly mean, but as far as you are concerned, you should consider them as synonymous. If you want to know/understand more, just come discuss these subtle differences with the teaching staff. Pour votre information, le runtime Java est parfois appelé le "Java Runtime Environment" ou bien encore la machine virtuelle Java ("Java Virtual Machine"). Mais pour ce qui vous concerne, ces noms sont essentiellement des synonymes. Si vous voulez en savoir plus, demandez aux enseignants.

We have provided you with an Eclipse already. It is important that you use the Eclipse we provided you. You will all be using Java projects that we started under the provided Eclipse and we do not want to run into configuration problems.

  • DO NOT INSTALL ECLIPSE VIA YOUR NORMAL INSTALLER.
  • DO NOT DOWNLOAD ECLIPSE FROM THE ECLIPSE WEBSITE.
  • DO NOT LAUNCH ECLIPSE BY CLICKING ON A DESKTOP ICON.
  • OR BY ANY OTHER MEANS, BUT THE ONE GIVEN BELOW.

We will launch Eclipse manually, from a terminal. This is really important, do follow exactly the instructions below. 

  ~$ cd ~/AgileLearning/workshops/oop
  oop$ source setenv.sh 
  oop$ eclipse -data workspace/

ALWAYS START ECLIPSE THIS WAY FOR THIS WORKSHOP.

Nice cookie

Maybe, you are annoyed as we are with your shell, because it is showing very long directory path, as part of its prompt. Like this: 

  ~$ cd ~/AgileLearning/workshops/oop
  ~/AgileLearning/workshops/oop$

And at some point, the prompting of the current directory takes an entire line. Well, if this is as annoying to you as it is to us, you can configure your shell to give you a shorter prompt. Under Ubuntu, using the standard shell, you can type the following command: 

  ~$ export PS1="\u@\h:\W$"

It takes effect only in the terminal you typed it in. If you want to make the change permanent, you need to add it to the startup file for your shell (~/.bashrc for the bash shell). Just overwrite the setting there.

If this cookie does not make sense to you, no biggie, just ignore it.

title=Object-Oriented Programming

Object-Oriented Programming

We are happy to have you as a new student and we hope that you will find this cursus interesting and motivating, but also challenging. The goal is to teach you object-oriented programming, using the Java programming language and tools. Nous sommes très heureux de vous compter parmi nos étudiants et nous espérons que vous trouverez cet atelier intéressant et motivant, mais aussi vous permettant de progresser. Le but est de vous apprendre la programmation orientée-objet, via le langage Java et les outils associés.

This course is structured following the principles of Agile Learning. Agile learning is a team effort, a team that encompasses the students as well as the teaching staff. Please read about these principles before proceeding. Ce cursus est organisé en suivant les principes de l'Apprentissage Agile. L'Apprentissage Agile est un apprentissage en équipe, un équipe qui inclue les étudiants mais aussi l'équipe enseignante. Veuillez lire les principes de cet apprentissage avant de vous lancer.

This workshop is organized in sprints and each sprint is composed of tasks. There is a progression to respect, so sprints have to be done in order and so are the tasks in each sprint. As you complete a task, the next task will be unlocked. As you complete all the tasks in a sprint, the next sprint will be unlocked. Cet atelier est organisé en sprints et chaque sprint est composé de tâches. Il y a donc une progression à respecter. Lorsque vous finirez une tâche, la suivante sera déverrouillée. De même, lorsque vous finirez un sprint, le suivant se déverrouillé.

Nota Bene: if you have finished a task or a sprint and you do not see the next one unlocked... do not panic and do not call for help. Just reload the page in your browser (hit Ctrl-R in most browsers) and the next task or sprint should now appear unlocked. If not, it means that you have not finished the previous task or sprint. Nota Bene: vous avez fini une tâche et la tâche suivante est toujours verrouillée? Où alors c'est le sprint suivant qui n'est pas déverrouillé? Pas de panique. Rechargez la page (avec la combinaison de touche Ctrl-R dans la plupart des browsers). La tâche ou le sprint suivant apparaîtra déverrouillée. Si ce n'est pas le cas, c'est que vous n'avez pas réellement fini la tâche ou le sprint en question.

Since you are reading this, you have already git-cloned this workshop and launch the local web server, via the shell script workshop.sh. You did all this from the directory ~/2018/workshops/oop, called the Workshop directory. Remember this, we will mention the Workshop directory throughout these pages. Puisque vous lisez ceci, vous avez déjà cloner via git cet atelier et lancer le server web local, en exécutant le script shell workshop.sh. Vous avez cela depuis le dossier ~/2018/workshops/oop, appelé le dossier de l'Atelier. Rappelez vous en car nous y ferrons référence souvent dans les pages qui suivent.

Using Shells

During this workshop, you will use shell terminals, as needed. Please, do not forget to setup properly the needed environment in each of these terminal. To do so, you need to source the file setenv.sh in each new terminal window. Durant cet atelier, vous allez utiliser des terminaux en ligne de commande. Rappelez vous de faire un "source" de setenv.sh, dans chaque nouveau terminal que vous allez utiliser pour cet atelier. 

  $ cd ~/AgileLearning/workshops/oop	  
  $ source setenv.sh
